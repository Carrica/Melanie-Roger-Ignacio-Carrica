Opa!
--------------------
¡Bienvenidos a Opa! Un sitio de ventas totalmente pensado
para cumplir todas las necesidades del consumidor al
mantener una interfáz y una estética dinámica y agradable
y al mismo una alta funcionalidad técnica.
Todos sabemos que no hay nada mejor que vivir experiencias 
increíbles que podamos vivir con nuestros amigos, familia o pareja, 
y mejor aún, compartirlas con el mundo entero a través del mundo digital, 
por eso en Opa! te ofrecemos varadas ofertas únicas para que cumplas tus sueños 
y generes memorias maravillosa para el resto de la vida. 


Guia de folder/archivos
-----------------------
Breve descripcion de los principales archivos y carpetas.
- resources
	- img // Contiene todas las imágenes usadas en el sitio.
	- html  // Contiene html alternos -No se usa-
- html
	-index
	-detalle
	-ingresar
	-registro
-js
	- datos.js // Posee los scripts para validar inicio de sesión
	- registro.js // Validación del registro
	- header.js // Le da animación al header, menu comprimido para pantalla de celular
	- pruebalistado.js // Ajax y jquery para los productos (arma dinámicamente el listado de ofertas, 
filtra según la categoría seleccionada, cuenta la cantidad de ofertas según su categoría para mostrarlo en el inicio)
	- armarDetalle // arma dinámicamente la información del producto seleccionado en el html detalle
-css
	- estilos.css // Hoja de estilo para todas las páginas
	- detalleEstilos.css // Hoja de estilo para la página detalle
